## 🌱 Mini Project 1 — Basic HTML Webpage

A simple webpage showcasing my favorite things such as:
- Food 🍕  
- Hobbies 🎨  
- Movies & Series 🎬  
- Travel 🇯🇵  

It also includes a simple recommendation form where users can suggest things for me.  
Users can select a category from a dropdown list such as:
Food recipes, movies, books, places to visit, and more.

### 🎯 Goal:
Understand the basic structure of a web page and essential HTML elements.

### 📚 Covered Topics:
- Basic structure of an HTML document
- Text and formatting
- Links and images
- Lists and tables
- Forms and input elements
